# صفحة هبوط لمقهى — وصف المشروع

ملخص
- صفحة هبوط احترافية لمقهى مصممة لعرض الهوية، القوائم، العروض، وخيارات التواصل أو الحجز. مناسبة للاستخدام كعارض أعمال على منصات العمل الحر.

المحتويات
- شيفرة مصدرية لواجهة المستخدم مبنية بـ Next.js
- أنماط جاهزة (Tailwind CSS + PostCSS)
- مكوّنات قابلة لإعادة الاستخدام (بطاقات، كاروسيل، ...)
- إعدادات جاهزة للنشر (Vercel/Node/static)

المميزات
- تصميم متجاوب وحديث مع دعم RTL بسهولة.
- أداء مُحسّن باستخدام Next.js.
- سهولة التخصيص للألوان، الخطوط، والمحتوى.

التقنيات المستخدمة
- Next.js (React)
- Tailwind CSS + PostCSS
- TypeScript (اختياري)

تشغيل محلي
```bash
npm install
npm run dev
# افتح http://localhost:3000
```

اقتراحات للتسليم على منصات العمل الحر
- العنوان: "تصميم صفحة هبوط لمقهى احترافية — Next.js + Tailwind"
- وصف مختصر للعرض:
  "أقوم بتطوير صفحة هبوط جاهزة للنشر لمقهى، تصميم نظيف ومتجاوب، مكوّنات قابلة لإعادة الاستخدام، مع دعم للتخصيص والنشر السريع."
- المخرجات التي تُسلم للعميل: المشروع الكامل (zip/repo)، تعليمات التشغيل، وعدد من جولات التعديل (مثلاً جولتين صغيرتين).
- وقت تقديري للتسليم: 2–4 أيام عمل حسب المتطلبات.
- نطاق سعري مقترح: 80–350 USD حسب مستوى التخصيص.

متى أحتاج من العميل

- قائمة المشروبات/المنتجات
- ألوان وهوية العلامة التجارية (إن وجدت)

ملاحظات الدعم
